Skynet - Getting Started

Getting Started

Prerequisites

You will need a computer with a C++ compiler with C++11 support or better (such as g++ or clang++ or vc++).
Makefiles are already includes with the project, so if you are running a Unix environment or have Cygwin/GitBash, you're all set.

Note, if you are using the lab machines in CHAP 103, none of this will work. GitBash is linking to G++ 4.6 which does not fully support C++11.
Cygwin is using an unpatched version of MinGW 4.9 which does not compile std::to_string.
This is as of 01/20/2016.

Download and Build

git clone https://github.com/uafcs600/skynet.git
cd skynet
make

Run Unit Tests

./unit_tests

Run Local Server (visit http://127.0.0.1:8080 in your web browser)

./server

Run the examples:

cd examples
evaluation/evaluation
info_game/info_game
list_games/list_games
move_generator/move_generator
play_game/play_game
